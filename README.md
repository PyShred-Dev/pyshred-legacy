# PySHRED

## Overview

PySHRED: A Python Package for SHallow REcurrent Decoders (SHRED) for Spatial-Temporal Systems

## Installation

1. Development Installation (Editable Mode):
```
pip install -e .
```

2. Standard Installation
```
pip install .
```
